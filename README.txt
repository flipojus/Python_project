Afin de lancer le programme, il faut lui donner en argument 
le fichier GeneSymbols_2.txt (ou autre fichier au même
format) comme ceci : 

python3 main.py GeneSymbols_2.txt

Il va de soi d'être dans le répertoire contenant les divers fichiers
.py et .txt. 

Toutes les infos seront affichées dans la sortie standard.

REMARQUE : Mon mail universitaire est utilisé pour recevoir les 
		   requêtes NCBI en cas de surchage, je n'ai pas jugé 
		   nécessaire de le rendre modifiable facilement (via une 
		   variable) à ce stade du projet.
